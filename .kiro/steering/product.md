# Product Overview

## MCP Stock Alert Agent

An automated stock and news monitoring system designed for swing trading and real-time market analytics. The system combines multiple components:

- **Automated Stock Analytics**: Periodic analysis of stock prices, news sentiment, and technical indicators
- **Real-time Alerts**: Email notifications for buy signals based on configurable criteria
- **Web Dashboard**: Live monitoring interface showing system health, top stocks, and market data
- **MCP Server Integration**: FastAPI WebSocket server for agentic communication
- **Enhanced Yahoo Finance**: Extended market data including earnings calendar and investment themes

## Target Use Case

Swing traders and investors who need:
- Automated screening of 80+ stocks across multiple sectors
- Technical analysis with RSI, volume, and price movement indicators
- News sentiment analysis from Yahoo Finance and Twitter/X
- Real-time market monitoring with customizable alerts
- Easy deployment across Mac/Linux systems

## Key Features

- **Multi-source Analysis**: Combines price data, news sentiment, and social media sentiment
- **Technical Indicators**: RSI calculation, volume analysis, price movement tracking
- **Smart Recommendations**: BUY/WATCH/NO SIGNAL based on multiple factors
- **Market Context**: Earnings calendar integration and investment theme tracking
- **Real-time Dashboard**: Web interface for system monitoring and stock performance
- **Portable Setup**: One-click deployment with shell scripts and virtual environments
